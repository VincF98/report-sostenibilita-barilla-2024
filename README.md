# Report di Sostenibilità Barilla 2024

Il presente repository contiene una pagina web realizzata a supporto del
Project Work universitario, dedicata alla presentazione del Report di
Sostenibilità Barilla 2024.

La pagina ha una funzione informativa e di accompagnamento al documento
ufficiale (report completo superiore alle dimensioni di upload massime, download per gli highlights), con l’obiettivo di rendere più immediato l’accesso ai contenuti
principali del report.

---

## Contesto del lavoro
Il progetto è stato sviluppato nell’ambito del Project Work per
l’Università Telematica Pegaso, all’interno del percorso di studi in ambito
economico–aziendale.

L’elaborato analizza il tema della sostenibilità d’impresa attraverso il
caso Barilla, soffermandosi sulle strategie, sugli impegni e sugli obiettivi
dichiarati dall’azienda.

---

## Obiettivo della pagina web
La pagina web ha lo scopo di:
- introdurre in modo sintetico il Report di Sostenibilità Barilla 2024;
- fornire un collegamento diretto al report completo;
- mettere a disposizione un documento di sintesi (highlights).

Si tratta di un supporto digitale complementare all’elaborato scritto.

---

## Struttura del repository
Il repository è composto dai seguenti file:
- `index.html`: struttura della pagina web;
- `style.css`: stile grafico e layout;
- `logo-barilla.png`: logo utilizzato nella pagina;
- `Highlights_report.pdf`: documento di sintesi del report.

---

## Tecnologie utilizzate
La realizzazione della pagina è avvenuta utilizzando:
- HTML5
- CSS3
- JavaScript

La pubblicazione online è stata effettuata tramite GitHub Pages.
