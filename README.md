# Report di Sostenibilità Barilla 2024

Questo repository contiene una pagina web dedicata alla presentazione del  
**Report di Sostenibilità Barilla 2024**, realizzata nell’ambito di un  
Project Work universitario.

---

## Obiettivo del progetto
L’obiettivo è realizzare una **landing page informativa** che:
- introduca il concetto di sostenibilità aziendale
- valorizzi il Report di Sostenibilità Barilla
- offra accesso diretto al report completo e a una versione di sintesi (highlights)

---

## Contenuto del repository
- `index.html` → struttura della pagina web  
- `style.css` → stile grafico e layout  
- `logo-barilla.png` → logo utilizzato nella pagina  
- `Highlights_report.pdf` → sintesi del report scaricabile  

---

## Tecnologie utilizzate
- HTML5
- CSS3
- JavaScript (per interazioni dinamiche)
- GitHub Pages (per la pubblicazione online)
